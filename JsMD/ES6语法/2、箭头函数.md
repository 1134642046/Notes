## 2、this

箭头函数的作用域是从当前位置向外一层一层找，直到找到一个this

```js {cmd}
const aa = {
  bb(){
    setTimeout(() => {
      console.log(this)
    })
  },
  cc(){
    setTimeout(function(){
      console.log(this)
    })
  }
}

aa.bb()
aa.cc()
```
