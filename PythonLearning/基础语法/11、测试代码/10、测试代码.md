# 测试代码

## 1、 测试函数

Python标准库中的模块unittest提供了代码测试工具。

### 1.1 单元测试和测试用例

下面是一个只包含一个方法的测试用例，它检查函数sum()在给定a和b时能否正确地工作：

```python {cmd .line-numbers}
import unittest

class SumTestCase(unittest.TestCase):
  def sum(a,b):
    return a+b

  def test_sum(self):
    result = sum(1,2)
    unittest.assertEqual(result,3)
unittest.main()


```
